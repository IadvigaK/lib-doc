TK01
Название:
Проверка авторизации пользователя (ввод валидных данных)

Шаги:
1. Перейти на сайт Alefowl.com
2. Нажать кнопку "Login" в правом верхнем углу
3. Заполнить поля "Login" и "Password" валидными данными (например, Login - haytovi4. Password - 123)
4. Нажать кнопку "Sign in"

Ожидаемый результат:
Пользователь авторизован на сайте Alefowl.com и попадает
на главную страницу с возможностью зайти в личный кабинет



TK02
Название:
Проверка авторизации пользователя (неверный логин)

Шаги
1. Перейти на сайт Alefowl.com
2. Нажать кнопку "Login" в правом верхнем углу
3. Заполнить поля "Login" не валидные данные и "Password" валидные данные (например, Login - haytovi5, Password - 123)
4. Нажать кнопку "Sign in"

Ожидаемый результат:
Пользователь не авторизован на сайте, появляется сообщение об ошибке



TK03
Название:
Проверка авторизации пользователя (неверный пароль)

Шаги:
1. Перейти на сайт Alefowl.com
2. Нажать кнопку "Login" в правом верхнем углу
3. Заполнить поля "Login" валидные данные и "Password" не валидные данные (например, Login - haytovi4, Password - 333)
4. Нажать кнопку "Sign in"

Ожидаемый результат:
Пользователь не авторизован на сайте, появляется сообщение об ошибке



ПРИМЕЧАНИЕ:
Поля Login, Password могут содержать любые символы. Ограничений по длине нет.
